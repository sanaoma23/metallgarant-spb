== Description ==

Плагин предназначен для передачи данных в сервис сквозной аналитик из форм созданных с помощью Contact Form 7 CoMagic (UIS) в качестве оффлайн-заявок. 
Для работы плагина требуется Contact Form 7 версии не ниже 4.6


= Docs & Support =

Поддержка работы плагина осуществляется в рамках технической поддержки сервиса CoMagic

= Privacy Notices =

Все что делает плагин, это передача данных в CoMagic согласно настройкам. 
Плагин не хранит никаких данных о пользователях или посетителях сайта, не собирает такие данные и не передает информацию никуда кроме API CoMagic.
Плагин не использует cookie или другие способы хранения информации в браузере

== Installation ==

Скпируйте папку плагина в директорию /wp-content/plugins. 
Затем активируйте плагин в веб-интерефейсе.
